I worked alone on developing my portfolio website, taking on the roles of both developer and designer. As a developer, I used HTML, SASS, and JavaScript to create the website's structure, styling, and interactive elements. As a designer, I designed the layout, selected the color scheme, and incorporated a space theme to reflect my creativity and interests.
The development process involved creating everything from scratch, as I hardcoded all elements of the website. I started with conceptualizing the layout and theme, then moved on to coding the HTML structure, styling it with SASS for a sleek and customized look, and adding interactivity using JavaScript where needed. I ensured consistency in design elements and responsiveness across different devices throughout the development process.
This project taught me a lot about solo web development and design. I gained hands-on experience in HTML, SASS, and JavaScript, honing my skills in frontend development. Designing the layout and choosing the color scheme helped me understand the importance of visual aesthetics in creating an engaging user experience. Additionally, hardcoding everything gave me a deeper understanding of how websites work at the code level and improved my problem-solving abilities in web development.
