What was my role in the project?

I filled in the roles of Web Designer, UX/UI Designer and Web Developer. The database is coded using React JS and the data is fetched from The Movie Database API. The styling was done with SASS.

What was the process?

At first I drew low-fidelity wireframes on paper to get a sense of where to place the images and sliders. Then I went on to create a high-fidelity mockup in Adobe Photoshop and a working prototype in Adobe XD.

The next step was to set up the project in React JS. [Say what files i decided to go with eventually]

To be fair, it was quite a challenge, there were many hiccups along the way, such as not being able to connect to the API at first. Fortunately, after that, I managed to quite easily fetch the desired data.

What have I learned?

I have learned how to work with APIs and how to make React components interact with each other.

I also gained a lot of confidence in working with frameworks with which I'm unfamiliar. Now, even if I have to complete a project using an unknown framework, language or library, I don't feel nearly as intimidated as I used to.
